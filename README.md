# Файлы
## Лысков Никита ИТ-8, 11 Вариант

Выполнить все задания в одном проекте в виде статических методов одного класса.
- В задании 1 и 2 бинарные файлы, содержат числовые данные, исходный файл необходимо заполнить
случайными данными, заполнение организовать отдельным методом.
- В задании 3 бинарные файлы содержат величины типа struct, заполнение исходного файла
необходимо организовать отдельным методом.
- В задании 4 в текстовом файле хранятся целые числа по одному в строке, исходный файл
необходимо заполнить случайными данными, заполнение организовать отдельным методом.
- В задании 5 в текстовом файле хранятся целые числа по несколько в строке, исходный файл
необходимо заполнить случайными данными, заполнение организовать отдельным методом.
- В задании 6 в текстовом файле хранится текст.

### 1
Найти количество квадратов нечетных чисел среди компонент файла

### Алгоритм решения
1. Прочитать все int из бинарного файла.
2. Для каждого числа: если оно нечётное, взять корень sqrt(num), привести к int и проверить, что root * root == num.
3. Считать такие числа и вывести их количество.


### 2
Скопировать элементы заданного файла в квадратную матрицу размером n×n (если элементов
файла недостает, заполнить оставшиеся элементы матрицы нулями). Указать строку (назвать её
номер), где максимальное количество элементов, кратных сумме индексов

### Алгоритм решения
1. Прочитать числа из бинарного файла и по очереди записать в матрицу n×n; недостающие элементы оставить 0.
2. Для каждой строки и каждого столбца посчитать сумму индексов sum = i + j.
3. Если sum > 0 и элемент matrix[i][j] делится на sum без остатка, увеличить счётчик для этой строки.
4. Найти строку, у которой этот счётчик максимальный, и вывести её номер.

### 3
Файл содержит сведения об игрушках: название игрушки, ее стоимость в рублях и возрастные
границы (например, игрушка может предназначаться для детей от двух до пяти лет). Получить
название самой дорогой игрушки для детей до четырех лет.

### Алгоритм решения
1. Прочитать числа из бинарного файла и по очереди записать в матрицу n×n; недостающие элементы оставить 0.
2. Для каждой строки и каждого столбца посчитать сумму индексов sum = i + j.
3. Если sum > 0 и элемент matrix[i][j] делится на sum без остатка, увеличить счётчик для этой строки.
4. Найти строку, у которой этот счётчик максимальный, и вывести её номер.

### 4
В файле найти среднее арифметическое максимального и минимального элементов

### Алгоритм решения
1. Прочитать текстовый файл построчно; каждая строка содержит одно целое число.
2. Преобразовать строку в число, обновляя minVal и maxVal.
3. После чтения вычислить (minVal+maxVal)/2.0 и вывести результат.

### 5
Вычислить произведение чётных элементов

### Алгоритм решения
1. Прочитать текстовый файл построчно; в каждой строке несколько целых чисел через пробел.
2. Для каждой строки с помощью std::istringstream по очереди извлекать числа.
3. Если число чётное, умножать его в переменную product и отмечать, что были чётные.
4. В конце, если чётных не было, вернуть 0, иначе вывести накопленное произведение.

### 6
Переписать в другой файл строки, в которых нет латинских букв

### Алгоритм решения
1. Прочитать текстовый файл построчно.
2. Для каждой строки проверить каждый символ: если это буква латинского алфавита (A–Z или a–z), пометить строку как содержащую латиницу.
3. Если латинских букв нет, переписать строку во второй файл и увеличить счётчик.
4. В конце вывести количество строк, переписанных без латинских букв.
